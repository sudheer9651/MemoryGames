# Number Memory Trainer

A lightweight static web application built with HTML, CSS, and vanilla JavaScript. Train your number memory with adjustable length and display duration, while saving settings and progress in local storage.

## Features

- Home screen with Start Training, Statistics, and Settings
- Number length options: 4, 6, 8, 10, 12, and custom length
- Display duration options: 3, 5, 10, 15 seconds
- Persistent settings via `localStorage`
- Training flow with countdown, hidden number, answer input, and feedback
- Total attempts, correct attempts, accuracy, current streak, and best streak
- Dark mode toggle
- Reset statistics button
- Keyboard Enter key submits answer
- Mobile-first responsive design with large buttons and smooth transitions

## Local usage

1. Open `index.html` in a modern browser.
2. Use the Settings screen to choose number length and display duration.
3. Click `Start Training`.
4. Memorize the number, then type it in and submit.
5. Review your performance in the Statistics screen.

## GitHub Pages deployment

1. Commit all files to your repository.
2. Push to GitHub.
3. Open repository Settings > Pages.
4. Select the `main` branch and root folder.
5. Save and visit the published URL.

Enjoy training your number memory!
